Auto Size Diagram.

In order to make diagram automatically size itself to the size of its content set pageFitMode property to AutoSize. This mode is designed for diagrams usage inside articals. Following properties allow to set minimum and maximum size of the diagram in this mode. Set minimum size to some value if you don't want it to collapse to nothing when diagram has no items. Set maximum size to some value if you need to avoid humongous diagram on your page.
